# GB/T 7714—2025 支持与迁移说明

## 标准状态

- 标准号：GB/T 7714—2025
- 名称：信息与文献 参考文献著录规则
- 发布日期：2025-12-02
- 实施日期：2026-07-01
- 被替代版本：GB/T 7714—2015

截至 2026-06-23，2025 版已经发布但尚未实施，2015 版仍是现行版本。本模板默认提前启用
2025 版；旧论文可将 `metadata.typ` 中的 `bibliography-standard-version` 改为 `"2015"`。

## 与 2015 版相比的技术变化

下表依据 GB/T 7714—2025 前言列出的“主要技术变化”整理。结构调整和纯编辑性修改不单列。

| 类别 | 2015 版 | 2025 版 |
| --- | --- | --- |
| 适用范围 | 原范围表述 | 重写范围，并明确适用于作者、编辑者编制参考文献，不适用于图书馆员、目录和索引编制者 |
| 术语 | 含“主要责任者”“专著”“数字对象唯一标识符”等 | 删除上述术语；新增“会议录”“数据集”“永久标识符”“预印本”“责任者”等，并修订多项定义和英文对应词 |
| 著录信息源 | 第 5 章规定 | 调整为第 4 章并重写表述 |
| 著录文字 | 原语种与汉语拼音人名规则 | 要求按信息资源本身语种著录；汉语拼音姓名宜按 GB/T 28039 拼写 |
| 著录符号 | 第 7 章 | 调整为第 6 章并重写符号适用范围 |
| 通则名称 | “著录细则” | 改为“著录通则” |
| 日期 | 公告日期、更新日期、引用日期 | 改为创建/发布或修改日期、引用日期；日期采用 `YYYY-MM-DD` |
| 标识符 | 数字对象唯一标识符（DOI） | 扩展为永久标识符；访问路径已含永久标识符时不重复著录 |
| 图书名称 | “专著”“专著中的析出文献” | 改为“图书”“图书中的析出文献” |
| 独立类型格式 | 多种类型沿用通用格式 | 新增会议录、学位论文、报告、标准、网站/网页、档案、地图、数据集、预印本的著录项目和格式 |
| 析出文献 | 单列通用规则 | 拆入图书、连续出版物等相应类型 |
| 著录项目 | 仅列元素 | 增加元素必备性和对应章条号 |
| 电子资源 | 单列通用格式 | 拆入各具体文献类型；电子资源同时著录文献类型和载体标识 |
| 引用体系 | “参考文献表”和“参考文献标注法”分章 | 合并为第 9 章“参考文献标引体系编制法” |
| 类型与载体代码 | 普通图书、舆图、EB 等旧称 | 改为图书、地图；新增预印本 `[PP]`；新增缩微资料 `/MM`；`EB` 明确表示网站、网页 |

## 对实现有直接影响的规则

1. 标点随文献语种选择，不是“2025 版全部改成中文全角标点”。中文条目使用中文逗号、冒号和圆括号，
   英文条目仍使用英文标点。
2. 2025 版英文人名保留连字符，例如 `Godard J-L`；超过 3 名责任者时保留前 3 名后加“等”或
   `et al.`。
3. 2025 版标准文献格式为“标准编号 标准名称 `[S]`”，不再输出责任者、出版地、出版者和出版年。
4. 报告以报告编号、发布日期为核心；会议录区分独立会议录和按图书析出文献出版的会议论文。
5. 网站、网页、数据集和预印本使用创建/发布或修改日期、引用日期、访问路径和永久标识符。
6. 顺序编码制重复引用页码置于文献序号方括号外；著者—出版年制重复引用页码置于圆括号外并以上角标著录。
7. 著者—出版年制参考文献表按中文、日文、西文、俄文、其他文种分组，再按责任者和出版年排序。

## BibTeX 建议

- 为每条文献填写 `language = {zh-CN}`、`en-US`、`ja-JP` 或 `ru-RU`，以便正确选择标点和排序分组。
- 非网页电子资源显式填写 `medium = {OL}`、`DK`、`CD`、`MT` 或 `MM`；仅有 URL/DOI 不自动推断载体。
- 特殊类型可使用 `mark`/`usera`：标准 `S`、网站/网页 `EB`、档案 `A`、地图 `CM`、数据集 `DS`、
  预印本 `PP`。
- 网站、网页、数据集和预印本建议同时填写 `date`、`urldate` 和 `url`。
- 永久标识符填写 `doi`；如果 URL 已经是对应 DOI 访问路径，2025 版不会重复输出 DOI。
